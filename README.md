# 🖤🤍 KSxKM – A Unique Platform for Developers & Digital Creators

**KSxKM** is a modern, stylish web platform created for developers, SMM specialists, and digital enthusiasts. Our goal is to build a central hub where tech lovers can share blogs, read the latest programming news, interact with each other, and showcase their services – all in one clean, black & white-themed interface.

---

## 🚀 Features

- 🔐 **Register & Login** – Secure authentication for all users, with separate profiles.
- 📰 **Blog System** – Write, view, edit, and delete blog posts. Add tags, likes/dislikes, and comments.
- 🗨️ **Commenting** – Registered users can comment on posts to engage in discussion.
- ❤️ **Like/Dislike System** – Express your thoughts about each post with simple feedback.
- 🏷️ **Categories/Tags** – Organize blog posts by topic or keyword.
- 🔍 **Search & Filter** – Easily find blogs based on title, tag, or author.
- ✉️ **Messaging** – Add friends and chat in real-time using local storage logic.
- 🌗 **Light/Dark Mode** – Seamless theme switching for better user experience.
- 🌍 **Multi-language Support** – Toggle between Uzbek, English, and more.
- 🛠️ **Fully Frontend-Based (for now)** – Using HTML, CSS, JavaScript + LocalStorage.
- 🧠 **Future Backend Support (Coming Soon)** – Node.js + MongoDB integration planned.
